# 電商 RFM 客戶價值分析儀表板

這是一個使用 Power BI 製作的客戶 RFM 分析儀表板，用來區分 VIP 客戶、一般客戶與沉睡客戶，協助行銷團隊制定精準行銷策略。

## 📊 儀表板預覽

![儀表板首頁](./images/dashboard-overview.png)
![客戶分群分析](./images/customer-segments.png)

## 🎯 分析維度

這個儀表板可以動態切換三種視角：

- **每日營收趨勢**：查看每日訂單數、營收與成長率
- **產品類別分析**：比較各產品類別的營收佔比
- **客戶類型對比**：新客戶 vs 舊客戶的消費行為


  ## 🛠️ 技術棧
- **語言**：Power BI ,SQL
- DAX 量值（動態切換、RFM 分群）
- 數據來源：模擬電商訂單數據（2024年11月）


## 📁 檔案說明

- `RFM_Dashboard.pbit` - Power BI 模板檔（不含數據，下載後可開啟）
- `/images` - 儀表板截圖

## 🚀 如何使用

1. 下載本倉庫的 `.pbit` 檔案
2. 用 Power BI Desktop 開啟（最新版本）
3. 載入你自己的數據（或使用範例數據）

## 📌 關鍵洞察

- VIP 客戶只佔 15%，卻貢獻了 45% 的營收
- 新客戶的平均客單價低於舊客戶，但成長空間大
- 電子產品類別營收最高，其次是服裝
